# きょうもがんばろう！ ⚔️🏰

こども向けの習慣トラッカーアプリです。ドラクエ風のダンジョンUIで、タスクをモンスターに見立てて「たおす」ことで達成していきます。

## 画面フロー

```
ぼうけんのしょ（初期画面）
    ├── しんきとうろく → メール・パスワード・あいことば入力 → 親としてログイン
    └── ログイン → メール・パスワード・あいことば入力 + 「おうちのひとですか？」チェック
                    ├── チェックあり → おうちのひと画面（タスク管理）
                    └── チェックなし → こども画面（タスク達成）
```

### 認証の仕組み

- **新規登録**: 親（おうちのひと）のみが行う。メールアドレス・パスワード・あいことばを登録
- **ログイン**: 親子共通。同じメール・パスワード・あいことばを使い、「おうちのひとですか？」チェックでロール分岐
- **こどもはログインのみ**: 親が登録した認証情報を使ってログインする

## ディレクトリ構成

```
help-children-app/
├── index.html              # HTMLエントリポイント
├── main.jsx                # Reactマウント
├── app.jsx                 # Appコンポーネント（セッション管理・画面切替）
├── constants.js            # 定数・DQカラーパレット・グローバルCSS
│                             - DQ: ドラクエ風カラー定義
│                             - TIMES: 時間帯（あさ/ひる/よる）の定義
│                             - css: DQ風アニメーション・フォント・UIクラス
├── store.js                # ストレージ操作（window.storage ラッパー）
│                             - registerUser / loginUser: ユーザー認証
│                             - getFamilyId / createFamily: 家族ID管理
│                             - getTasks / saveTasks: タスクのCRUD
│                             - getCompletions / addCompletion / resetCompletions: 達成記録
├── storage-mock.js         # ローカル開発用 window.storage モック
├── vite.config.js          # Vite設定
├── package.json            # 依存関係
│
└── components/
    ├── ui.jsx              # 共通UIパーツ（DQ風ウィンドウ）
    │                         - LoginCard / Section / STitle / TimeChip / Loader
    ├── Login.jsx           # ログイン・新規登録画面
    │                         - 初期画面: 新規登録 or ログイン選択
    │                         - 新規登録: メール+パスワード+あいことば → 親ログイン
    │                         - ログイン: メール+パスワード+あいことば+親チェック
    ├── Parent.jsx          # おうちのひと画面（タスク管理・しんちょく確認）
    └── Child.jsx           # こども画面（タスク達成・レベル/EXP表示）
```

## UIテーマ: ドラクエ風ダンジョン 🐉

- **DotGothic16** ピクセルフォント
- ダンジョン風の暗い背景（`#0A0A1A`）
- 青枠の RPG ウィンドウ（`#4A8FD4` ボーダー）
- 金色テキスト（`#FFD700`）でタイトル・アクセント
- タスク = モンスター、達成 = 「たおした！」
- こども画面にレベル・EXP バー表示
- `▶` カーソル付きの DQ 風コマンドボタン

## 開発コマンド

| コマンド | 説明 |
|---------|------|
| `npm install` | 依存パッケージをインストール |
| `npm run dev` | 開発サーバー起動（http://localhost:5173/） |
| `npm run build` | プロダクションビルド |
| `npm run preview` | ビルド結果のプレビュー |

## 技術スタック

- **React 19** — UIライブラリ
- **Vite 6** — ビルドツール・開発サーバー
- **window.storage** — データ永続化（本番はプラットフォーム提供、ローカルは localStorage モック）
